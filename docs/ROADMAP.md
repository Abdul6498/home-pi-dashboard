# Roadmap

## Phase 1 (MVP: Clock + Weather + GPS)

1. Full-screen themed dashboard shell
2. Real-time clock/date rendering
3. GPS location resolution with fallback coordinates
4. Weather API integration and resilient display states
5. Systemd service for auto-start on Raspberry Pi

## Phase 2 (Smart home usefulness)

1. Calendar cards
2. Reminders + notifications
3. Home automation quick actions
4. Day/night adaptive theme switching

## Phase 3 (Polish + reliability)

1. Offline caching and retries
2. Config UI screen
3. Performance tuning for Pi 3 GPU/CPU constraints
4. Telemetry/logging for module health
